VECTOR

Vector is a set of numbers that can identify a point in space.

UNIT VECTOR

Unit vector is a vector with a unit norm (norm is a function that assigns positive length to a vector).

ORTHOGONAL VECTORS

Vector x and vector y are orthogonal (perpendicular) to each other if (their dot product equals 0). If both vectors have nonzero norm and their dot product equals 0, then they are perpendicular to each other.

ORTHONORMAL VECTORS

Vector x and vector y are orthonormal if they are orthogonal and have unit norm.

VECTOR NORM

Norm is a tool that allows us to measure the size of a vector (a non-negative number). It has a form of different functions, suitable for different situations. The norm is defined such that:

MANHATTAN NORM

This norm is calculated as a sum of absolute values in a vector.

EUCLIDEAN NORM

This norm is calculated as a root of a sum of squares of all values in a vector.

MAX NORM

This norm is calculated by taking a maximum absolute value of a vector.



VECTOR ADDITION

When we add two vectors, we get a resulting vector which represents a motion in the direction of the first vector and then the second vector (or vice versa).

DOT PRODUCT

Dot product (scalar product) is an operation that takes 2 equal-length vectors and returns a number.


In geometric interpretation, a dot product of vectors a and b is a length of a projection a onto b multiplied by length of b.


Why do we need it?
1. Given two vectors, lets us test if these vectors are orthogonal (perpendicular to each other).
2. Given two vectors, lets us find the length of one vector in the direction of another.
